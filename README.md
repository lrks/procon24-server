Procon24 Server
===============
「第24回 高専プロコン」の競技部門の練習に使えるサーバです。自由部門の人が作りました。

* 問題文、トークン、競技時間の設定
* 回答の送信
* 試合状況の確認
* 試合状況の実況

が可能です。


使い方
------
node.js と socket.io をインストールした上で、`node server.js`でサーバを起動します。

`http://(IPアドレス):3000/`にアクセスしてください。

`http://(IPアドレス):3000/SubmitForm`と
`http://(IPアドレス):3000/SubmitAnswer`
というページも用意してあります。
